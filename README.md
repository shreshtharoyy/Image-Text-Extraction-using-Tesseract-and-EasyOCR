# 📄 OCR using Tesseract & EasyOCR

## 📌 Overview

This project demonstrates basic Optical Character Recognition (OCR) using:

* PyTesseract (Tesseract OCR)
* EasyOCR

Text is extracted from a sample receipt image and displayed as output.

---

## 🎯 Purpose

This project is created for **learning purposes** to understand:

* How OCR works
* How different OCR libraries extract text
* Basic implementation of OCR in Python

---

## 📂 Demo Input

A sample receipt image is used for demonstration.

> Note: This project uses a single image for simplicity instead of the full dataset.

---

## ⚙️ Setup

```bash
pip install pytesseract Pillow easyocr opendatasets
sudo apt-get install tesseract-ocr
```

---

## 🚀 How it works

1. Load the image
2. Extract text using:

   * Tesseract OCR
   * EasyOCR
3. Print the extracted text

---

## 📊 Output

The notebook displays OCR results directly in the output cells.

---

## ⚠️ Notes

* This is a **basic implementation for learning**
* Only a single sample image is used
* Outputs are generated when the notebook is executed
